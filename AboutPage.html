Loci

My Senior project is a VR memory tool called Loci, which uses the HTC Vive to help users create mind palaces. Mind palaces are an old memory technique, where a person remebers a sequence of items by mentaly placing them in a familiar building, and visualizing a walkthrough. Coming to the Steam store spring 2018.

Conky

I wrote a personal configuration of the conky system monito for my Linux laptop based off of conky-lunatico. The system monitor displays temperature and usage of system resources, shows a spotify now playing, and features an RSS feed.
